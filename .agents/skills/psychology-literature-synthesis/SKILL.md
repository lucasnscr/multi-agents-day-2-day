# Psychology Literature Synthesis

Summarize psychology research on behavior, decision-making, stress, habits, motivation, and communication.

## Activation

Use this skill when the user asks about: psychology, behavior, motivation, stress, decision-making, emotion.

## Workflow

1. Clarify whether the task is education, self-reflection, or workplace communication.
2. Use reputable sources and distinguish evidence, interpretation, and personal reflection prompts.
3. Translate concepts into practical exercises while avoiding diagnosis.
4. Escalate crisis, self-harm, abuse, or clinical concerns to qualified support.

## Expected Outputs

- literature summary
- reflection prompts
- practical exercise plan

## Safety And Quality Boundaries

- Never diagnose, treat, or replace a licensed mental health professional.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
