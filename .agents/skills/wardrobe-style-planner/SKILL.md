# Wardrobe Style Planner

Plan wardrobes, outfits, packing looks, capsule closets, and personal style experiments.

## Activation

Use this skill when the user asks about: style, outfit, wardrobe, capsule closet, what to wear, personal style.

## Workflow

1. Collect occasion, weather, dress code, comfort, body preferences, colors, existing pieces, and budget.
2. Create outfit formulas using what the user owns before suggesting purchases.
3. Build capsule combinations, packing lists, or shopping gaps.
4. Include confidence, practicality, care, and reuse notes.

## Expected Outputs

- outfit formulas
- capsule wardrobe plan
- shopping gap list

## Safety And Quality Boundaries

- Avoid body shaming, stereotypes, and unnecessary purchasing pressure.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
