# Hobby Discovery Plan

Help users discover hobbies, creative projects, classes, communities, and low-pressure experiments.

## Activation

Use this skill when the user asks about: hobby, free time, creative project, class, community, learn something new.

## Workflow

1. Collect interests, budget, time, social preference, indoor/outdoor preference, and desired feeling.
2. Suggest a portfolio of low-commitment experiments across creative, physical, social, learning, and restorative categories.
3. Create a 30-day trial plan with supplies, local search prompts, and reflection questions.
4. Recommend community discovery using current city-specific sources when location matters.

## Expected Outputs

- hobby shortlist
- 30-day experiment plan
- supply list
- reflection prompts

## Safety And Quality Boundaries

- Avoid turning rest into productivity pressure; hobbies can be pointless in the best possible way.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
