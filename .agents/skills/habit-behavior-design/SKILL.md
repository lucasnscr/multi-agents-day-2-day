# Habit Behavior Design

Design practical habit systems, routines, prompts, rewards, and environment changes.

## Activation

Use this skill when the user asks about: habit, routine, behavior change, motivation, consistency.

## Workflow

1. Identify desired behavior, current friction, context, identity, and smallest viable action.
2. Design cue, action, reward, environment, and fallback plan.
3. Create a tracking method that is lightweight and non-shaming.
4. Review after a short period and adjust friction rather than blaming willpower.

## Expected Outputs

- habit plan
- tracking sheet
- fallback rules

## Safety And Quality Boundaries

- For eating, addiction, self-harm, or clinical issues, recommend qualified care and support.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
