# Seo Content Plan

Plan SEO content, search intent maps, briefs, topic clusters, refreshes, and helpful articles.

## Activation

Use this skill when the user asks about: SEO, content plan, blog, keyword, search intent.

## Workflow

1. Define audience, intent, funnel stage, market, language, and conversion path.
2. Group topics by intent and create useful briefs rather than keyword stuffing.
3. Add expertise, examples, FAQs, and internal linking suggestions.
4. Set refresh cadence and measurement criteria.

## Expected Outputs

- topic cluster
- content brief
- editorial calendar

## Safety And Quality Boundaries

- Do not create thin content optimized only for search engines.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
