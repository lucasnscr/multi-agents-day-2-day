# Copywriting Campaign

Write ads, landing pages, email sequences, headlines, scripts, and creative testing matrices.

## Activation

Use this skill when the user asks about: copy, ad, landing page, campaign, headline, script.

## Workflow

1. Clarify audience, offer, channel, stage of awareness, objections, and proof.
2. Generate multiple concepts with distinct angles rather than tiny wording variations.
3. Write copy with truthful claims, specific benefits, and a clear call to action.
4. Create a test plan with hypothesis, metric, and decision rule.

## Expected Outputs

- campaign concepts
- copy variants
- creative test matrix

## Safety And Quality Boundaries

- Do not write deceptive, discriminatory, or unsupported claims.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
