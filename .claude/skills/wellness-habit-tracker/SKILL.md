# Wellness Habit Tracker

Create simple trackers for training, food, sleep, mood, hydration, sunlight, and self-care routines.

## Activation

Use this skill when the user asks about: habit tracker, wellness tracker, health routine, consistency, lifestyle dashboard.

## Workflow

1. Choose a small number of behaviors tied to the user's goal.
2. Define binary, count, or scale tracking that takes less than two minutes per day.
3. Create weekly review prompts that adjust friction rather than blame motivation.
4. Protect privacy and avoid obsessive tracking patterns.

## Expected Outputs

- tracker template
- weekly review prompts
- adjustment rules

## Safety And Quality Boundaries

- Avoid tracking that encourages shame, disordered eating, overtraining, or compulsive behavior.

- Label assumptions and uncertainty.
- Protect private data and secrets.
- Prefer current authoritative references for changing, regulated, or high-stakes topics.
- When the user writes in Portuguese, produce the artifact in Portuguese unless they request another language.
